# 课堂展示设计原则（五区浮层布局）

本文档定义课堂展示的布局标准，用于指导后续所有新建演示页面。

## 1. 设计目标

- 让教师在 1080P 大屏上快速进入"可讲状态"。
- 3D 动画全屏铺底，核心控件和数据以浮层/侧栏形式覆盖，不抢占动画空间。
- 移动端（≤767px）自动切换为 viewport + 底部面板布局。

### 1.1 运行时边界

- 课堂模式契约建立在现代 Chromium 教学环境上（`Worker` + `WebGL2` + `Pointer Events`）。
- 本文档不定义弱环境降级策略；能力不足由应用统一"运行环境不支持"阻断页处理。

## 2. 五区布局架构

```
┌──────────────────────────────────────────────────┐
│  ┌─────────┐                        ┌──────────┐ │
│  │ 参数控制 │                        │ 数据区   │ │
│  │ (侧栏)   │    动画区 (3D)          │ (悬浮)   │ │
│  │         │     全屏底层 z-0        └──────────┘ │
│  │ 折叠 ←→ │                                    │
│  │ 320/44px │         ┌──────────────┐            │
│  │         │         │  图表区(悬浮) │            │
│  │         │         │  可关闭       │            │
│  │         │         └──────────────┘            │
│  └─────────┘                                    │
│              ┌──────────────────────┐            │
│              │ 播放控制区(悬浮)      │            │
│              │ ▶ ⏸ ⏪ 速度控制       │            │
│              └──────────────────────┘            │
└──────────────────────────────────────────────────┘
```

### 区域属性

| 区域 | 默认状态 | 位置 | 折叠行为 | 可关闭 |
|---|---|---|---|---|
| 动画 | 始终可见 | 全屏底层 z-0 | — | 否 |
| 参数 | 展开 | 左侧栏 320px (z-30) | 折叠为 44px 窄条 + 展开按钮 | 否 |
| 数据 | 展开 | 右侧悬浮 (z-10) | 折叠为标题栏 | 否 |
| 图表 | 按需打开 | 右侧悬浮 (z-10) | 折叠为标题栏 | 是 |
| 播放 | 展开 | 底部悬浮 (z-20) | 折叠为标题栏 | 是 |

## 3. 核心原则

1. **3D 优先**：动画区始终全屏铺底，其他区域以浮层覆盖，不分割视口。
2. **参数可折叠**：侧栏折叠后仅占 44px，教师可快速隐藏参数区让学生聚焦动画。
3. **数据随时可见**：数据浮层始终展开，实时读数不被折叠隐藏。
4. **图表按需**：图表浮层默认关闭，需要时从数据区打开，不需要时可关闭释放空间。
5. **触控友好**：所有交互元素最小 44px 触控目标，符合 WCAG 2.5.5。

## 4. 新场景接入清单

1. 为 `SceneLayout` 传入 `controls`（参数滑块/选择器）、`dataOverlay`（实时读数）、`chart`（波形/图表）、`playbackActions`（播放控制）。
2. 保持分层结构：`useXxxSceneState` + `XxxControls` + `XxxRig3D` + `XxxScene` 壳层。
3. 确保 `dataOverlay` 包含课堂关键读数（3~5 行），教师折叠侧栏后仍可从数据浮层获取核心信息。
4. 图表组件应支持 `chartVisible` 控制显隐。
5. 播放控制通过 `playbackActions` 数组传入，移动端自动提升到 header。

## 5. 移动端适配（≤767px）

- header 隐藏标题，只显示 [返回] + [播放按钮] + [昼夜切换]
- viewport 占满剩余空间
- 底部面板固定 30vh，通过 SegmentedControl 切换 参数/数据/图表 tab
- 播放控制不再单独显示，已提升到 header

## 6. 常见反模式（禁止）

- 把核心读数放在折叠面板深处且无其他区域可见替代。
- 让教师必须多次点击才能看到当前结论。
- 新场景不提供 `dataOverlay`。
- 浮层按钮使用小于 44px 的触控目标。
